# 커뮤니티 로고 교체 폴더

상단바(홈) 브랜드 로고를 여기서 바꿀 수 있습니다.

## 교체 방법
- 이 폴더의 **`logo.svg`** 파일을 원하는 로고로 교체하세요(같은 파일명 유지).
- PNG/JPG를 쓰려면 파일을 넣은 뒤 `public/community/home.html`의
  `<img src="assets/logo/logo.svg?v=7" ...>` 에서 `logo.svg` 부분을 파일명으로 바꾸세요.
- 권장: 정사각형, 투명 배경(브랜드 인디고 칸 위에 표시됨). 너무 복잡하지 않은 단순 심볼.

## 캐시 주의
- 교체 후에도 옛 로고가 보이면 `home.html`의 `?v=7` 숫자를 올려서(예: `?v=7`) 배포하거나
  Cloudflare 캐시를 퍼지하세요. (`/community/**` 는 no-store 헤더라 보통 새로고침으로 반영됩니다.)

## 로고를 못 불러오면
- 이미지 로드 실패 시 자동으로 기본 텍스트 "유"가 표시됩니다(깨지지 않음).
